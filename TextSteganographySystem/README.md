# Text Steganography System

A Java security-based application that hides secret text messages inside a cover text file without changing its visible appearance.

---

# Features

- Accept cover file and secret message
- Encode and hide secret message
- Save encoded output file
- Retrieve hidden message
- Preserve visible content
- Support multiple test cases

---

# Concepts Used

- Java
- String Processing
- Unicode Characters
- File Handling
- Security Concepts
- Exception Handling

---

# How to Run

## Compile

```bash
javac TextSteganography.java
```

## Run

```bash
java TextSteganography
```

---

# Working Principle

- Convert secret message into binary
- Convert binary into invisible Unicode characters
- Hide invisible data inside cover text
- Retrieve and decode hidden data
