# Java_Innovative_Projects

A collection of innovative Java projects developed for academic learning and practical implementation of core Java concepts such as object-oriented programming, collections, file handling, security concepts, and string processing.

---

# Projects Included

## 1. Smart Password Generator
A Java application that generates secure passwords based on user-defined strength and requirements.

### Features
- Secure random password generation
- User-defined password strength
- Uppercase, lowercase, numbers, and symbols support
- Password strength evaluation
- Uses SecureRandom for better security

---

## 2. Personal Expense Tracker
A Java-based application used to manage and track daily expenses.

### Features
- Add and store expenses
- Generate monthly reports
- Display all expense records
- Find highest expense category
- File handling support

---

## 3. Text Steganography System
A security-based Java project that hides secret messages inside a text file without changing its visible content.

### Features
- Hide secret messages
- Retrieve hidden messages
- Invisible Unicode-based encoding
- File handling implementation

---

# Technologies Used

- Java
- Collections Framework
- File Handling
- OOP Concepts
- String Processing
- Security Concepts
- LocalDate API
