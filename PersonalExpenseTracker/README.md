# Personal Expense Tracker

A Java-based application developed to manage and track daily expenses efficiently.

---

# Features

- Add daily expenses
- Store amount, category, and date
- Display all expenses
- Generate monthly expense reports
- Identify highest expense category
- Save and load expense data using file handling

---

# Concepts Used

- Object-Oriented Programming
- ArrayList
- HashMap
- File Handling
- Serialization
- LocalDate API
- Exception Handling

---

# How to Run

## Compile

```bash
javac ExpenseTracker.java
```

## Run

```bash
java ExpenseTracker
```

# Functionalities

- Expense Recording
- Monthly Expense Analysis
